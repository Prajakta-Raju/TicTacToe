# Tic Tac Toe

A responsive and interactive Tic Tac Toe game built using **HTML**, **CSS**, **JavaScript**, and **jQuery**. This project recreates the classic two-player game with real-time turn switching, winner detection, animated winning lines, sound effects, and a clean user interface.

##  Features

* Two-player gameplay (X vs O)
* Automatic turn switching
* Winner detection across rows, columns, and diagonals
* Animated winning line visualization
* Sound effects for moves and game events
* Victory GIF animation
* Reset button to start a new game
* Responsive design for desktop and mobile devices
* Smooth hover effects and transitions

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* jQuery

##  How It Works

1. Players take turns clicking on empty cells.
2. The game places either **X** or **O** in the selected box.
3. After every move, the game checks all possible winning combinations.
4. When a player wins:

   * A winning message is displayed.
   * A line is drawn through the winning combination.
   * A celebration GIF appears.
5. Players can reset the board at any time using the **Reset** button.
